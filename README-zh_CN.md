Ant Design 3.X 的 Vue 实现，开发和服务于企业级后台产品。


[README in English](README.md)

## 特性

- 提炼自企业级中后台产品的交互语言和视觉风格。
- 开箱即用的高质量 Vue 组件。
- 全链路开发和设计工具体系。 

## 安装

### 使用 npm 或 yarn 安装

```bash
$ npm install vue-antd-ui --save
```

```bash
$ yarn add vue-antd-ui
```

如果你的网络环境不佳，推荐使用 [cnpm](https://github.com/cnpm/cnpm)。
