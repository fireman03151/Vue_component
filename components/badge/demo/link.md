<us>
#### Clickable
  The badge can be wrapped with `a` tag to make it linkable.
</us>

```html
<template>
  <a-badge dot>
    <Icon type="notification"></Icon>
  </a-badge>
  <a-badge dot>
    <a href="#" class="head-example">Link something</a>
  </a-badge>
</template>
```
